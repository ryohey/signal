# PipeDream 2.0

> A simple and enjoyable music creation tool with AI assistance. Create music through an intuitive interface where AI helps you compose.

**Powered by [Signal](https://signalmidi.app)**

---

## Features

- **AI-assisted song creation** with human-in-the-loop editing
- **Intuitive MIDI editing** with piano roll and arrange views
- **Real-time playback** and synthesis
- **Cross-platform** web application
